# Kanban Board Overview

The **Kanban Board** component helps organize and visualize project tasks across
different workflow stages like **To Do**, **In Progress**, and **Done**.

## Props

| Prop | Type | Description |
|------|------|-------------|
| `tasks` | `KanbanTask[]` | Tasks to display in board columns |
| `columns` | `string[]` | Column names for board |
| `onTaskMove` | `(id: string, status: string) => void` | Called when a task moves |
